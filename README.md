# Foundations & Algorithmic Analysis

A comprehensive collection of fundamental **Data Structures and Algorithms (DSA)** programs focused on **algorithmic thinking, recursion, complexity analysis, and problem-solving techniques**.
This repository contains practical implementations, theoretical explanations, complexity evaluations, and recursion tracing for important foundational concepts in computer science.

---

## 📚 Topics Covered

### 1. Stack ADT: Design + Correctness + Meaningful Use

Implementation of the **Stack Abstract Data Type (ADT)** using standard operations:

* Push
* Pop
* Peek / Top
* isEmpty
* Display

Includes:

* Stack design principles
* Correctness verification
* Real-world applications of stack
* Time complexity analysis

#### Concepts Learned

* LIFO (Last In First Out)
* Memory management
* Function call behavior
* Expression handling

---

### 2. Complexity Drill: Operation Counting + Best/Average/Worst Case

Programs focused on analyzing algorithm performance using:

* Operation counting
* Time complexity measurement
* Best case analysis
* Average case analysis
* Worst case analysis

Includes:

* Step-by-step execution analysis
* Growth rate comparison
* Big-O notation understanding

#### Concepts Learned

* Efficiency comparison
* Performance evaluation
* Algorithm optimization basics

---

### 3. Recursive Factorial + Call Stack Trace + Complexity

Recursive implementation of factorial with detailed recursion tracing.

Features:

* Recursive function execution
* Call stack visualization
* Recursive breakdown explanation
* Complexity analysis

#### Concepts Learned

* Base case and recursive case
* Function call stack
* Recursion depth
* Time complexity of recursive algorithms

---

### 4. Fibonacci: Naive vs Memoized + Call Counter Comparison

Comparison between:

* Standard recursive Fibonacci
* Memoized Fibonacci approach

Includes:

* Function call counting
* Performance comparison
* Dynamic programming introduction
* Optimization using memoization

#### Concepts Learned

* Overlapping subproblems
* Recursive inefficiency
* Memoization techniques
* Dynamic programming basics

---

### 5. Tower of Hanoi: Recursion Trace (N=3) + Complexity Pattern

Implementation of the classic **Tower of Hanoi** problem with recursive tracing.

Features:

* Step-by-step disk movement
* Recursive tree understanding
* Complexity pattern visualization

#### Concepts Learned

* Divide and conquer strategy
* Recursive decomposition
* Exponential complexity behavior

---

### 6. Recursive Binary Search + Recurrence Intuition

Efficient searching technique using recursion.

Includes:

* Sorted array searching
* Midpoint division logic
* Recursive recurrence intuition
* Complexity analysis

#### Concepts Learned

* Divide and conquer
* Logarithmic complexity
* Recursive searching
* Efficient algorithm design
